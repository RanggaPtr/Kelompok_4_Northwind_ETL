# Kelompok_4_Northwind_ETL
Northwind ETL &amp; Data Warehouse Project — Kelompok 4 Repository ini berisi implementasi lengkap proses ETL (Extract–Transform–Load) dan Data Warehouse menggunakan dataset Northwind sebagai studi kasus Business Intelligence. Proyek mencakup pembuatan star schema, transformasi data menggunakan Pentaho Data Integration, serta Menggunakan Looker.
